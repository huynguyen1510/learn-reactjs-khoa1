Tạo 1 project trên github
1/ Tạo 1 github repository(folder) - nơi chứa code
2/ Clone project này - git clone https://github.com/huynguyen1510/learn-reactjs-khoa1.git
3/ Viết code
4/ git status (có thể bỏ qua)
5/ git add .
6/ git commit -m "Viết comment để ghi nhớ"
7/ git push origin main
